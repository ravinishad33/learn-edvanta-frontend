Edvanta🎓
A modern, animated Learning Management System built with React, Tailwind CSS, and Framer Motion (MERN).

🚀 Live Demo
Coming Soon

✨ Features
🔐 User Authentication
Role-based registration (Student/Instructor/Admin)

Social login integration

Email/OTP verification

Secure password management

📊 Role-Based Dashboards
Student Dashboard: Course progress, certificates

Instructor Dashboard: Course creation, student analytics, earnings

Admin Dashboard: User management, platform analytics, system monitoring

🎓 Course Management
Interactive course catalog with filtering

Course creation wizard for instructors

Rich media support (videos, PDFs, presentations)

Curriculum builder with modules and lessons

📱 Learning Experience
Interactive video player

Progress tracking with visual indicators

Discussion forums per course

Certificate generation upon completion

🎨 UI/UX Features
Smooth animations with Framer Motion

Fully responsive design

Dark mode ready

Accessible components

Loading states and skeletons

🛠️ Tech Stack
React 18 - Frontend framework

Tailwind CSS - Styling

Framer Motion - Animations

React Router DOM - Navigation

Heroicons - Icon library

Chart.js - Data visualization
